函数

函数是什么？通俗来说 y = f(x),是将一个对象（x) 转换成另一个对象（y）的规则，而规则就是f。我们把起始对象（x）称为输入，把返回的对象称为（y），x有来自叫定义域的集合，而y有叫来自上域的集合

  • 例如
    当你写出形式 ：y=x2y=x^2y=x2，这就定义了一个函数，x是输入对像，y是输出对象，而f(x)=yf(x)=yf(x)=y，我们说x通过f这个规则 得到了y。所以，当我们输入x = 2，那么y就会=4，简单的运算。
    我们不能平白无故的随便输入东西进去，比如y=f(马)y=f(马)y=f(马)，你不可能把马平方了，是不。我们把这种不会得出结果和结果没有意义的输出，称为没有定义,因为他不在定义域里面 则引出了一个定义
一个函数必须给每一个有效的输入指定一个唯一的输出
那么我们怎么处理把马平方这种情况呢？

什么？你不会以为我真的要把马平方吧。不不不，假设我们限制能输入的东西 让它能通过这种规则输出有意义的值，我们把这种方法叫做限制定义域
我们把x的输入限制在（只包含0和大于0的数）（我们叫做非负数），也就是说除了小于0的数，你都能输入，并且对于每一个x都有一个对应的值。

现在记住，如果你定义了一个函数，要列出定义域的范围

值域是所有可能输出的集合，你可以认为值域是函数转变定义域的一切，它可能会有重复，但这没什么。上域其实是所有可能输出的集合，而值域其实是上域的一个子集，值域是实际输出的集合。

y=x2y=x^2y=x2,其定义域和上域是所有实数(用R表示)，其值域是所有非负数的，当你平方一个数的时候，它的结果不可能是负的。例如-2和2，他们的结果都是4.不可能说-2的平方结果是-4，或者其他的负数。

区间表示法

形如[a,b] 是指a端点到b端点间所有的实数集(其中也包括a和b)。(a,b)
他也包含a到b所有的实数集，但是和[a,b]不同的是，它不包含a和b两个点。我们说:

形如[a,b]的我们叫闭区间，形如(a,b)的我们叫开区间

如果是介于a和b之间，但是包含b不包含a的呢，我们记作(a,b]，形如这种的，我们叫做半开区间。
表示一个区间的方法除了这种，还有描述法
形如

{x∈R: 2 < x < 5} 等同于(2,5)

形如{x∈R ：2 ≤ x ≤ 5}， 它等同于[2,5]

  • 一种特殊的情况
    有的集合表示为（a，∞),这里要注意，无穷大不是一个实数，所以你不能写成(a,∞]。因为你不可能在数轴上找到它，这是不可能的，所以，我们认为，无穷大不是一个实数，没有一个数等于它。
